# Issue #23: Design Lexicon for dataset index records

## Verdict: CLOSE

## Evidence
The `ac.foundation.dataset.record` lexicon is fully implemented as `LexDatasetRecord` in the codebase. It supports hybrid storage via an open union of three storage types: `StorageHttp` (HTTP/HTTPS URLs with per-shard checksums), `StorageS3` (S3/S3-compatible with bucket, region, endpoint), and `StorageBlobs` (ATProto PDS blob references). The record includes name, schema reference, storage, metadata (structured `DatasetMetadata` with legacy msgpack backward compat), tags, size info, license, and timestamps. `DatasetPublisher` provides methods for all storage variants.
