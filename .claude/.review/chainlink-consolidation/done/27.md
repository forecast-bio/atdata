# Issue #27: Implement schema publishing to ATProto

## Verdict: CLOSE

## Evidence
`SchemaPublisher` and `SchemaLoader` are fully implemented. `SchemaPublisher.publish()` introspects a PackableSample class, extracts field definitions via `get_type_hints()` and `dataclasses.fields()`, maps Python types to JSON Schema properties, and publishes the result as an `ac.foundation.dataset.schema` ATProto record. `SchemaLoader` fetches schema records by AT URI with type validation, supports typed access via `get_typed()` returning `LexSchemaRecord`, and lists all schemas from a repository. The schema body uses JSON Schema Draft 7 with NDArray shim support.
