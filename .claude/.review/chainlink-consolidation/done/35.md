# Issue #35: Add caching and indexing for performance

## Verdict: ARCHIVE

## Evidence
Caching has been implemented at the module level throughout the codebase rather
than as a dedicated caching layer. Key caching mechanisms include: `_type_cache`
in `_schema_codec.py` for dynamically generated types with LRU eviction,
`_pds_endpoint_cache` in `atmosphere/__init__.py` for PDS endpoint resolution,
`lru_cache` decorators on lexicon loading functions, and
`_sample_type_cache` on `Dataset` instances for type parameter resolution.
Additionally, a full benchmark suite exists for measuring and tracking
performance, with CI integration for regression detection.
